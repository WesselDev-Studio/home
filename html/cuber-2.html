Cuber 2: Ice Age

About The Game

Cuber 2: Ice Age is the sequel of the game Cuber. Cuber 2: Ice Age is a 2D platformer game with enemies shooting at the player and with ice.

The main goal is to get to the end of levels and jump into the portal. Side goals are killing the enemies via bombs and collecting coins to score points.

Where To Play The Game

Cuber 2: Ice Age is played on Windows(32 or 64 bit), MacOS and Linux(32 or 64 bit). Download the correct version for your system.

For Windows run the .exe file and install it, and run it via shortcut.

For MacOS and Linux extract the .zip file and in the folder run the nw file.
